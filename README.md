Aula 03 – Controlando o Back End Nesta aula iremos mostrar como colocar um template em HTML, CSS e JS para funcionar em um Back end NodeJS. Utilizaremos a biblioteca Express para criar um servidor e as rotas da nossa aplicação. E iremos utilizar o EJS para a template engine, ou seja, a ferramenta capaz de gerar o HTML e CSS pelo back-end.

Aula 04 – Persistindo os dados Chegou o momento de conhecer o SQL, o que é um banco de dados relacional e por que é importante para nossa aplicação. Você vai aprender a salvar e não perder mais os dados da sua aplicação. Vamos criar os Models para avançar na nossa aplicação.
